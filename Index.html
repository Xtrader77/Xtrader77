Compounding Logic

Win = level up, Loss = level down, Break Even = repeat level

Outcome Action Result
Win (TP Hit) Move up to next level Bigger lot size next trade
Loss (SL Hit) Go down one level Smaller lot size next trade
Break Even Repeat current level Same lot size next trade
